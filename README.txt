Свадебное приглашение
=====================

Открыть на компьютере: дважды щёлкните index.html

Папка: c:\Users\ПК\Documents\wedding-invite

При открытии — экран с конвертом. Нажмите красную печать.

Изменить в index.html: имена, даты, адрес, телефон, тексты.
Изменить в script.js: WEDDING_DATE для обратного отсчёта.


КАК СДЕЛАТЬ ССЫЛКУ ДЛЯ ТЕЛЕФОНА
================================

Сайт должен лежать в интернете. Локальный файл по ссылке не отправить.

СПОСОБ 1 — САМЫЙ ПРОСТОЙ (Netlify Drop, бесплатно)
--------------------------------------------------
1. Откройте в браузере: https://app.netlify.com/drop
2. Перетащите всю папку wedding-invite на страницу
   (внутри должны быть: index.html, styles.css, script.js, папка images)
3. Сайт выдаст ссылку вида: https://какое-то-имя.netlify.app
4. Эту ссылку отправляйте гостям в WhatsApp, Telegram, SMS
5. На телефоне ссылка откроется в браузере как обычный сайт

Можно задать своё имя в настройках сайта на Netlify (например: alina-sasha-wedding.netlify.app)


СПОСОБ 2 — GitHub Pages (бесплатно, если есть аккаунт GitHub)
-------------------------------------------------------------
1. Создайте репозиторий на github.com
2. Загрузите все файлы из папки wedding-invite
3. Settings → Pages → Source: main branch → Save
4. Ссылка: https://ваш-логин.github.io/имя-репозитория/


СПОСОБ 3 — хостинг (Reg.ru, Timeweb и т.д.)
-------------------------------------------
Загрузите файлы в папку public_html на хостинге через файловый менеджер или FTP.


ВАЖНО ПРИ ЗАГРУЗКЕ
------------------
- Загружайте ВСЮ папку целиком, особенно images\sakura-bg.jpg
- index.html должен быть в корне сайта (не в подпапке)
- После загрузки откройте ссылку на телефоне и проверьте конверт и фон


УДОБНО ДЛЯ ГОСТЕЙ
-----------------
- QR-код со ссылкой (создаётся на netlify или на сайтах qr-code-generator)
- Короткая ссылка: bit.ly / clck.ru — если длинный адрес неудобен

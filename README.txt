Свадебное приглашение
=====================

Открыть на компьютере: дважды щёлкните index.html

Папка: c:\Users\ПК\Documents\wedding-invite

При открытии — экран с конвертом. Нажмите красную печать.

Изменить в index.html: имена, даты, адрес, телефон, тексты, фото (блок .hero__photo).
Изменить в script.js: WEDDING_DATE для обратного отсчёта.


КАК СДЕЛАТЬ ССЫЛКУ ДЛЯ ТЕЛЕФОНА
================================

Сайт должен лежать в интернете. Локальный файл по ссылке не отправить.

СПОСОБ 1 — GitHub Pages (рекомендуется)
---------------------------------------
Пример: https://hramuhov-novice.github.io/wedding-invite/

1. Репозиторий на github.com с файлами из папки wedding-invite
2. Settings → Pages → Branch: main, папка / (root) → Save
3. Ссылка: https://ваш-логин.github.io/имя-репозитория/
4. После правок: загрузите файлы снова или git push — сайт обновится за 1–3 мин.


СПОСОБ 2 — Netlify Drop (бесплатно, без GitHub)
-----------------------------------------------
1. https://app.netlify.com/drop — перетащите папку wedding-invite
2. Ссылка: https://какое-то-имя.netlify.app


СПОСОБ 3 — хостинг (Reg.ru, Timeweb и т.д.)
-------------------------------------------
Загрузите файлы в папку public_html на хостинге через файловый менеджер или FTP.


ВАЖНО ПРИ ЗАГРУЗКЕ
------------------
- Загружайте ВСЮ папку целиком, особенно images\sakura-bg.jpg
- index.html должен быть в корне сайта (не в подпапке)
- После загрузки откройте ссылку на телефоне и проверьте конверт и фон


УДОБНО ДЛЯ ГОСТЕЙ
-----------------
- QR-код со ссылкой (создаётся на netlify или на сайтах qr-code-generator)
- Короткая ссылка: bit.ly / clck.ru — если длинный адрес неудобен
